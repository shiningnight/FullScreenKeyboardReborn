# 完全屏幕键盘重生

## 简介

使用驱动级模拟的屏幕键盘。较之系统自带的屏幕键盘，特有持续按键锁定、一键连发等按键辅助功能，专为单手鼠标玩游戏优化，也可作为游戏辅助工具。可载入Comfort On-Screen Keyboard Pro的键盘布局文件，基本兼容其布局特性。

DD虚拟鼠标键盘驱动: https://github.com/ddxoft/master
